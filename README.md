# CK IMMO FINANZ - Dashboard

KI-gestütztes Prozess-Automatisierungssystem für Immobilienfinanzierung.

## Features

- 11-Phasen Pipeline für Finanzierungsprozesse
- Kunden-Management mit Status-Ampel
- Team-Übersicht
- Automatisierungs-Log
- WhatsApp-Integration (Twilio)
- KI-Dokumentenklassifizierung

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (Auth + DB)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
